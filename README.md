# Functional Programming

1. No Loops
1. No Ifs
1. Function is a Single Return
1. No Side-Effects
1. No Assignments in Functions
1. No Arrays
1. Only Functions with 0 or 1 Arguments

## FizzBuzz Problem. 

- input: `[1, 2, 3, 4, 5, 6, 7, ..., 100]`
- output: `["1", "2", "fizz", 4, "buzz", "fizz", "7", ..., "buzz"]`
